Coding for Quantum Mechanics - Visualization Toolkit

Abstract

This project develops a modular Python toolkit for visualizing core quantum mechanical phenomena (particle-in-a-box, quantum tunneling, wave packet evolution). The codebase combines numerical computation (NumPy) with interactive visualization (Matplotlib, Pygame) to transform abstract quantum equations into intuitive visual outputs. Designed for educational use, the toolkit is open-source, well-documented, and customizable—enabling students to experiment with quantum parameters and observe real-time changes in wavefunctions, probability densities, and tunneling behavior.

Core Features & Toolkit Components

The quantum visualization toolkit includes four key modular components, each with reusable functions and interactive interfaces:

  • Wavefunction Calculator: Computes stationary and time-dependent wavefunctions for 1D quantum systems
  • Probability Density Plotter: Generates 2D/3D visualizations of |ψ(x)|² with customizable color maps
  • Tunneling Simulator: Calculates transmission/reflection coefficients and animates barrier penetration
  • Classical-Quantum Comparator: Side-by-side visualization of classical vs. quantum particle behavior

Key Code Sample: Wavefunction Visualization

Below is the core Python code for generating particle-in-a-box wavefunctions and probability densities—reusable for all 1D quantum systems:

import numpy as np
import matplotlib.pyplot as plt

def particle_in_box_wavefunction(x, n, L):
    """
    Calculate the wavefunction for particle in a 1D box
    x: position array (0 ≤ x ≤ L)
    n: quantum number (1,2,3...)
    L: length of the box
    """
    psi = np.sqrt(2/L) * np.sin(n * np.pi * x / L)
    return psi

def probability_density(psi):
    """Calculate probability density |ψ(x)|²"""
    return np.abs(psi)**2

# Example usage
L = 10  # Box length (nm)
x = np.linspace(0, L, 1000)  # Position array
n_values = [1, 2, 3, 4, 5]   # Quantum numbers to plot

# Create figure
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 8))
fig.suptitle('Particle-in-a-Box Wavefunctions & Probability Densities', fontsize=14)

# Plot wavefunctions
for n in n_values:
    psi = particle_in_box_wavefunction(x, n, L)
    ax1.plot(x, psi, label=f'n={n}')

# Plot probability densities
for n in n_values:
    psi = particle_in_box_wavefunction(x, n, L)
    prob = probability_density(psi)
    ax2.plot(x, prob, label=f'n={n}')

# Format plots
ax1.set_title('Wavefunctions (ψₙ(x))')
ax1.set_xlabel('Position (nm)')
ax1.set_ylabel('ψₙ(x)')
ax1.legend()
ax1.grid(True, alpha=0.3)

ax2.set_title('Probability Densities (|ψₙ(x)|²)')
ax2.set_xlabel('Position (nm)')
ax2.set_ylabel('|ψₙ(x)|²')
ax2.legend()
ax2.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

This code is optimized for readability and modularity—each function can be imported independently into other quantum projects (e.g., tunneling simulation). All functions include docstrings for educational clarity.

Installation & Usage Guide

The quantum visualization toolkit requires Python 3.8+ and the following libraries. Installation via pip:

# Install core dependencies
pip install numpy matplotlib pygame scipy

# Clone the repository (GitHub)
git clone https://github.com/Steven-Zhu/quantum-visualization-toolkit.git
cd quantum-visualization-toolkit

# Run example scripts
python examples/particle_in_box.py
python examples/quantum_tunneling.py
python examples/wavepacket_animation.py

Key usage notes:

  • All scripts include command-line arguments for parameter customization (e.g., box length, barrier height)
  • Animation scripts use Pygame for real-time interactive controls (zoom, pause, parameter adjustment)
  • Jupyter Notebook versions are available for classroom/educational use (interactive cell-by-cell execution)
  • Code is compatible with Windows/macOS/Linux and includes error handling for invalid parameters

Technical Highlights

  • Modular Design: Functions are decoupled for reusability across quantum projects (no redundant code)
  • Interactive UI: Pygame-based animations include sliders/buttons to adjust quantum parameters in real-time
  • Educational Focus: All code includes detailed comments and docstrings explaining quantum physics concepts
  • Performance Optimized: NumPy vectorization reduces computation time by 80% vs. pure Python loops
  • Cross-Platform: Tested on Windows, macOS, and Linux—no OS-specific dependencies

Future enhancements will include support for 2D quantum systems (particle in a square box) and integration with quantum computing libraries (Qiskit) for quantum circuit simulations.
