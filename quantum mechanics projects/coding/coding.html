Computational Quantum Mechanics Projects

Abstract

This coding project develops a suite of Python-based computational tools to visualize core quantum mechanical phenomena (wavefunctions, probability densities, tunneling, time evolution). The goal is to translate abstract quantum equations into interactive, visual outputs (graphs, animations) that make quantum mechanics accessible to a broader audience. The codebase is modular, well-commented, and optimized for educational use, with future plans to deploy a web-based interface (Flask/Dash) for browser-based interaction without Python installation.

Fundamental Principles (Computational)

The computational approach to quantum visualization is grounded in numerical solutions to the Schrödinger equation and classical computational physics principles:

  • Time-Independent Schrödinger Equation (TISE): Solved numerically for bound states (particle in a box) using finite difference methods: -ℏ²/(2m) d²ψ/dx² + V(x)ψ = Eψ.
  • Time-Dependent Schrödinger Equation (TDSE): Simulated for wave packet evolution using split-operator methods to preserve unitarity (probability conservation).
  • Numerical Integration: NumPy's vectorized operations are used to efficiently compute wavefunction values, avoiding slow loop-based calculations.
  • Visualization Principles: Matplotlib's animation API is used to render time evolution, with frame rates optimized for smooth playback and educational clarity.

The code prioritizes physical accuracy (correct units, quantum constants) while maintaining readability for students learning both quantum mechanics and computational physics.

Technical Stack

Core Technologies

  • Python (core language) - chosen for readability and scientific computing libraries
  • NumPy - vectorized numerical calculations (10-100x faster than pure Python loops)
  • Matplotlib - static plots and basic animations (time evolution)
  • Seaborn - enhanced visualization styling for publication-quality plots
  • Pygame - interactive wave packet simulations (real-time user input)
  • Jupyter Notebooks - educational documentation (code + explanations)
